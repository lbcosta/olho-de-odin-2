# Olho de Odin 2

Olho de Odin é uma aplicação que roda localmente (local first) e se conecta a API externa do servidor LATAM para obter dados de mercado do jogo Ragnarok Online. O objetivo principal é auxiliar o jogador a tomar decisões na hora de vender seus itens, fornecendo informações sobre preços, estoque, concorrência e estratégias recomendadas.
